# 生活费预算支配 - 宣传主页

这是一个独立的宣传网页项目，采用 React + Vite + Tailwind CSS + Framer Motion 构建

## 🚀 如何运行

1. 进入目录：
   ```bash
   cd landing-site
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

3. 启动开发服务器：
   ```bash
   npm run dev
   ```

4. 构建生产版本：
   ```bash
   npm run build
   ```

## 🌐 部署到 GitHub Pages

本项目已配置 `base: './'`，构建后的 `dist` 文件夹可以直接部署到 GitHub Pages 或任何静态网站托管服务。

1. 运行构建命令：
   ```bash
   npm run build
   ```

2. 将 `dist` 文件夹内的所有文件上传到你的 GitHub 仓库的 `gh-pages` 分支，或者直接在 Settings -> Pages 中指向该文件夹（如果作为独立仓库）。
